# KIT: Keep In Touch

## PRM: Personal Relationship Manager

KIT: Keep In Touch application helps you keep up with the people in your life. Birthdays, Anniversaries and other special events should not go unnoticed. This web app will keep you up-to-date on upcoming events with your personal and proffesional contacts by prioritizing and display the need for following up with that contact. 

# Features Include
1. Adding contacts by Family, Friends or Professional
2. Call reminder to family.
3. Up-Coming Event reminders
4. Contact Lists
5. Phone List
6. Direction Links to Contacts Home
7. Call Links from the app
8. Email links
9. Sort by personal type
